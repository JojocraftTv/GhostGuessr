# GhostGuessr Steam Installation Guide

This guide will help you install GhostGuessr for the Steam version of GeoGuessr.

## Prerequisites

- GeoGuessr installed via Steam
- Windows operating system

---

## Installation Steps

### Step 1: Download the Patcher

1. Go to the [Releases page](https://github.com/JojocraftTv/GhostGuessr/releases/latest)
2. Download the latest **GhostGuessr-Patcher.exe**
3. Save it to a location you can easily find (e.g., Downloads folder)

### Step 2: Run the Patcher

1. Open `GhostGuessr-Patcher.exe`
3. If Windows SmartScreen appears, click **"More info"** → **"Run anyway"**

### Step 3: Apply Patch

1. Click **"Patch"** button
2. Wait for the process to complete

### Step 4: Launch GeoGuessr

1. Close the patcher
2. Launch GeoGuessr from Steam
3. GhostGuessr features should now be available!

---

## Updating

You don't need to repatch it, it will be updated automatically.

---

## Uninstalling

### Method 1: Using the Patcher

1. Run the patcher again
2. Look for **"Uninstall"** button
3. Click it to remove GhostGuessr

### Method 2: Verify Game Files

1. Open Steam
2. Right-click **GeoGuessr** in your library
3. Select **Properties** → **Installed Files** → **"Verify integrity of game files"**
4. Steam will restore the original files

##  Troubleshooting

### Patcher won't run?

- ✓ Temporarily disable antivirus (some may flag the patcher)
- ✓ Check Windows Defender exclusions

### Can't find GeoGuessr folder?

1. Open Steam
2. Right-click GeoGuessr → **Manage** → **Browse local files**
3. This will open the installation folder
4. Copy the path from the address bar

### Patch failed?

- ✓ Close GeoGuessr completely (including background processes)
- ✓ Verify you have write permissions to the folder

### Features not showing in game?

- ✓ Restart GeoGuessr completely
- ✓ Re-run the patcher to ensure installation succeeded
- ✓ Check if Steam updated the game (may need to re-patch)

---

## Important Notes

- **Steam Updates**: When GeoGuessr updates through Steam, you may need to re-run the patcher
- **Backup**: Consider backing up your game files before patching
- **Antivirus**: Some antivirus software may flag the patcher as suspicious (false positive)
