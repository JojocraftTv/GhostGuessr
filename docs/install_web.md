## Chrome or Edge

### Step 1: Install Tampermonkey

First, you need to install Tampermonkey if you haven't already.

1. Go to the [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
2. Click **"Add to Chrome"** (or "Add to Edge" if using Edge)
3. Click **"Add extension"** when prompted
4. The Tampermonkey icon will appear in your browser toolbar

### Step 2: Enable Developer Mode

1. Open Chrome/Edge and navigate to `chrome://extensions/` (or `edge://extensions/`)
2. Toggle **"Developer mode" ON** in the top right corner
3. Keep this tab open for reference

### Step 3: Install GhostGuessr Script

1. Click this install link: **[Install GhostGuessr](https://raw.githubusercontent.com/JojocraftTv/GhostGuessr/refs/heads/main/script.user.js)**
2. Tampermonkey will open automatically
3. Click **"Install"** to confirm
4. The script is now installed and active!

### Step 4: Verify Installation

1. Navigate to [GeoGuessr](https://www.geoguessr.com/)
2. You should see GhostGuessr features enabled
3. If not, refresh the page or check Tampermonkey settings

## Firefox

### Step 1: Install Tampermonkey/Greasemonkey

1. Go to the [Firefox Add-ons Store](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
2. Click **"Add to Firefox"**
3. Click **"Add"** when prompted
4. The extension icon will appear in your toolbar

## Troubleshooting

### Script not working?

- ✓ Make sure Tampermonkey is enabled
- ✓ Check that the GhostGuessr script is enabled in Tampermonkey dashboard
- ✓ Try refreshing the GeoGuessr page
- ✓ Clear your browser cache and reload

### Tampermonkey icon not visible?

- ✓ Click the puzzle piece icon in your toolbar
- ✓ Pin Tampermonkey to make it always visible

### Still having issues?

- Open an issue on the [GitHub repository](https://github.com/JojocraftTv/GhostGuessr/issues)
